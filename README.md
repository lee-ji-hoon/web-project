### MYSQL 세팅
- CREATE DATABASE aqua_project;
- CREATE USER aqua@'%' IDENTIFIED BY '1234';
- GRANT ALL PRIVILEGES ON aqua_project.* TO aqua@'%' WITH GRANT OPTION;
